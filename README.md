# pokedexapp -
## приложение, с помощью которого можно погрузиться в мир покемонов:
* Список покемонов:  
    * Изображение покемона
    * Его имя  
    * Стихия покемона   
* Профиль покемона:  
    * Крупное изображение покемона   
    * Подробные характеристики  
    * Стадии эволюции    
* История эволюции покемона:  
    * Слабости покемона    
    * Физические показатели покемона
